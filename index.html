🏛️ 성균관대학교
대학별 세부 주제: 사회 갈등/통합
퍼실리테이터 / 갈등조정전문가
💼 당신이 퍼실리테이터라면?

지역개발 갈등 현장에서 이해관계자들의 의견을 분석하고, 합의점을 도출하며, 중재 시뮬레이션을 진행하는 역할을 수행합니다.

📌 프로젝트 예시
AI 텍스트 분석으로 쟁점 도출 갈등조정 챗봇 개발 합의안 자동 생성 도구
🌟 가천대학교
대학별 세부 주제: 삶의 질 향상
웰니스 기획자 / 공공정책 분석가
💼 당신이 웰니스 기획자라면?

지역주민의 생활 데이터를 분석하여 삶의 질을 높이는 정책을 제안하고, 우선순위를 설정하여 실행 가능한 웰빙 프로그램을 기획합니다.

📌 프로젝트 예시
QoL 지표 분석 대시보드 맞춤형 웰빙 추천 시스템 정책 우선순위 제안서
🌾 충남대학교
대학별 세부 주제: 지역활성화
로컬 크리에이터 / 도시재생 코디
💼 당신이 로컬 크리에이터라면?

소멸위기 지역의 숨은 매력을 발굴하고, AI를 활용해 관광 콘텐츠를 제작하며, 유휴공간을 활성화하는 창의적인 전략을 수립합니다.

📌 프로젝트 예시
AI 관광 추천 시스템 상권 분석 대시보드 공간 활성화 시나리오
🌏 건양대학교
대학별 세부 주제: 다문화
다문화 교육 기획자 / 사회복지 매니저
💼 당신이 다문화 교육 기획자라면?

다문화 가정의 정보 접근성을 높이고, 언어 장벽을 해소하며, 문화적 이해를 증진시키는 교육 콘텐츠와 서비스를 개발합니다.

📌 프로젝트 예시
다국어 FAQ 챗봇 문화 교육 콘텐츠 생성 정보 접근성 개선 앱
👴👶 한동대학교
대학별 세부 주제: 저출산·고령화 대응
시니어 케어 기획자 / 인구정책 연구원
💼 당신이 시니어 케어 기획자라면?

독거노인의 삶의 질을 향상시키고, 세대 간 소통을 촉진하며, 저출산 문제에 대한 데이터 기반 정책 대안을 제시합니다.

📌 프로젝트 예시
AI 대화 동반자 챗봇 세대 연결 플랫폼 정책효과 시뮬레이션
